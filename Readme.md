# Docker Commands For Setting Up Postgres and Redis

Setup Postgres
`docker run --name postgres -e POSTGRES_PASSWORD=postgres -d -p 5432:5432 postgres`

Setup Redis
`docker run -p 6379:6379 -it redis/redis-stack-server:latest`
